
# 🌲 Everforest ncspot port

A beautiful dark theme for [ncspot](https://github.com/hrkfdn/ncspot) - a Spotify client for the terminal. This is a port of the popular [Everforest](https://github.com/sainnhe/everforest) color scheme.

## Features

- 🎨 **Warm, nature-inspired colors** from the Everforest medium palette
- 🌙 **Easy on the eyes** with carefully chosen contrast ratios
- 🎵 **Optimized for ncspot** interface elements
- 📦 **Simple installation** and configuration

## Installation

### Prerequisites

- [ncspot](https://github.com/hrkfdn/ncspot) installed and configured

### Steps

1. Clone this repository:
   ```bash
   git clone https://github.com/GuilhermeTerriaga/everforest-ncspot.git 
   ```

2. Save the theme configuration to: ~/.config/ncspot/config.toml 

3. If you're using ncspot on flatpak, move to: ~/.var/app/io.github.hrkfdn.ncspot/config/ncspot/
